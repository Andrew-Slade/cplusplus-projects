
Namespaces:

------------------
namespaces are libraries.

reference used to access namespaces is using namespace <namespace>;
this is called using a directive

cout is a class.

using can also reference multiple objects:

using std::cout;
using std::cin; these are called declarations

no more using namespace std; 
must use declarations;
